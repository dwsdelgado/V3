---
title: "TIL: docker ps --filter ancestor filtra por imagen base"
date: 2026-06-02
tags: ["docker", "til"]
---

Hoy aprendí que antes de eliminar una imagen Docker puedes ver qué contenedores la usan:

```bash
docker ps -a --filter ancestor=nginx:latest
```

El flag `-a` incluye contenedores detenidos, no solo los que están corriendo. Si no aparece nada, la imagen está libre para borrar.

```bash
docker rmi nginx:latest
```

Si hay contenedores detenidos que la referencian primero hay que borrarlos:

```bash
docker rm nombre-contenedor
docker rmi nginx:latest
```

Simple pero no lo sabía, siempre hacía `docker rmi -f` a lo bruto.
